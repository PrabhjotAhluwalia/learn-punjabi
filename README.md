# ਸਤਿ ਸ੍ਰੀ ਅਕਾਲ — Learn Punjabi

A free, comprehensive single-file web app for learning Punjabi (Gurmukhi) from scratch — built for native English speakers with zero prior exposure.

**Live site:** see GitHub Pages (Settings → Pages) and the pplx.app link.

## What's inside
- **Start Here** — intro to Punjabi, the 130M+ speaker community, Gurmukhi vs Shahmukhi, a 5-level path, and a progress tracker.
- **Script** — all 41 Gurmukhi letters with audio, stroke order, the ਕ vowel-combination row, matras, special signs, a Trace-It grid, and a romanization quiz.
- **Lessons** — 40 lessons across 10 levels, each with vocabulary, a grammar concept, example sentences, and a quiz.
- **Phrasebook** — 200+ phrases by category with literal meanings and formality tags.
- **Vocabulary** — 500+ words across 17 themes, with flashcards and spaced repetition.
- **Grammar** — 10 sections plus verb conjugation tables.
- **Practice** — 5 games: Script Speed Quiz, Translation Challenge, Listening Drill, Fill-in-the-Blank, Word Scramble.
- **Culture** — geography (with a live Leaflet map), food, music, festivals, diaspora, proverbs, and the language family tree.

## Tech
Single self-contained `index.html`. React 18, Tailwind, Framer Motion, Lucide, Leaflet (all via CDN), and the browser Web Speech API for audio. No build step, no backend, no tracking. Progress saved in `localStorage`.

## Run locally
Just open `index.html` in any modern browser.
